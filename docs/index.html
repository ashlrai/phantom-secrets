Three steps. Zero code changes.

Works underneath your existing workflow. Nothing to learn, nothing to configure.

1

Protect your secrets

Run phantom init. Real secrets move to an encrypted vault (OS keychain or ChaCha20-Poly1305 file). Your .env is rewritten with worthless phm_ tokens.

2

Code with AI normally

Run phantom exec -- claude. A local proxy starts on 127.0.0.1. AI reads your .env, sees only phantom tokens. Your workflow doesn't change.

3

Everything just works

When code calls an API, the proxy swaps each phantom token for the real credential and forwards over TLS. Your code works. The AI never knew.
